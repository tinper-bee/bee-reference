react bee-reference component for tinper-bee

参照组件 `bee-reference`，支持固定格式的数据进行模式化选择。

## 使用方法

```js
$ yarn add bee-reference
```

```
import Reference from 'bee-reference'

ReactDOM.render(<Reference />, appRoot)
```
